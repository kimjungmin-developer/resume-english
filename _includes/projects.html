DI Framework 만들기

2019.10.30 ~ 2019.11.21
https://github.com/kimjungmin-developer/di-framework


페어프로그래밍, Spring의 DI기능을 모방하여 의존주입 해주는 Framework 구현

  • - Spring과 DI를 체계적으로 공부하기 위해서 그룹스터디 꾸림
    1. 그룹스터디에서 학습했던 전략패턴/템플릿-콜백패턴에 감명을 받은 후 적극적으로 코드에 활용하여 이해도를 높임

  • - 사용자의 편의성을 위하여 재귀함수를 이용하여 순환참조 감지 구현
    1. 위 기능을 구현하면서 서비스의 품질은 개발자의 작은 정성이 모여 완성된다는 것을 배움
    2. 언급한 순환참조를 제대로 감지하는지 검증하기 위해 테스트 코드 구현

  • - 사용자의 편의성을 위하여 생성자주입을 Annotation없이 가능하게 구현
    1. 스프링이 마법같이 해주던 것을 직접 구현해보면서 생성자 주입의 원리를 파악
    2. 원리를 파악하여 Spring/DI Framework에 대한 이해도 증진
    3. 스프링에서 제공하는 기능을 최대한 자신만의 방식으로 구현하기 위해 고민하고 실행
    4. 새로 배운 개념을 잘적용하고 있는지 확인하기 위해서 다양한 테스트케이스 추가
    5. 테스트 케이스를 만들면서 미처 생각하지 못한 버그를 발견하면서 테스트코드의 유용성을 다시 한번 확인
